# hayaken-shop — ホームセンター通販ECサイト（デモ）

ホームセンター通販を参考に構築した EC サイトのデモ実装。**単独の Next.js 15（App Router）アプリ**で、外部のモノレポやパッケージには依存しません。

商品・カート・注文などのデータはすべて **モックデータ＋ブラウザの localStorage** で完結し、外部サービス（DB・決済・認証）には接続していません。

## 主な機能

### 商品・カタログ
- **12カテゴリ・88商品**のモックカタログ（DIY・工具／金物・建材／塗料／園芸／カー・自転車／掃除／キッチン／収納／ペット／作業安全／家電／日用品・防災）
- 商品名から**サブカテゴリを自動分類**（中カテゴリ）
- 在庫切れ商品の表示（グレーアウト・購入無効化・入荷お知らせ）
- 商品画像は外部アセット不要の**SVGプレースホルダ**（カテゴリ色＋絵文字）で生成

### 探す・回遊
- トップ：**ヒーロースライダー**（自動送り）／特集帯・特集一覧／カテゴリ／売れ筋・セール・新着／**あなたへのおすすめ**（閲覧履歴ベース）／最近チェックした商品
- ヘッダー**検索サジェスト**（カテゴリ＋商品候補）、検索ページの人気キーワード
- カテゴリページ：**サブカテゴリ・ブランド・価格帯・セール**の多段絞り込み＋並び替え＋**ページネーション**
- **特集ページ** `/features/[slug]`（DIYフェア／園芸／防災／新着）
- **商品比較**（最大4件・固定比較バー・スペック横並び）

### 商品詳細
- 価格・在庫・ポイント・数量選択・お気に入り・比較
- **説明／仕様／レビュー**のタブUI
- カスタマーレビュー（評価分布＋口コミ）と**レビュー投稿**（localStorage）
- 関連商品・最近チェックした商品

### カート・購入
- カート（localStorage永続化・数量変更・送料計算）
- **送料無料までの進捗バー**・空カート時の**おすすめ表示**
- **クーポン**（`/coupons` 一覧・コピー・適用）
- **ポイント**（獲得／利用充当・残高管理）
- **複数配送先（分割配送）**：商品ごとに住所を割当て、配送先ごとに送料計算
- **配送日時指定・ギフト設定**（のし・メッセージ）
- 注文完了→注文番号・獲得/利用ポイント表示・**注文確認メールのプレビュー**

### 会員（マイページ）
- 注文履歴・**注文詳細**（配送先別内訳・ステータスタイムライン）
- **注文キャンセル**（ポイント巻き戻し）・**再注文**
- **お届け先の管理**（住所録の追加・編集・削除）
- 保有ポイント・お気に入り集計

### 管理画面（`/admin`）
- **ダッシュボード**（KPI・売上グラフ・最近の注文・在庫アラート）
- **商品管理**（在庫・価格のその場編集／CSVエクスポート）※ストア表示に即時反映
- **在庫アラート**（在庫切れ・在庫わずか）
- **クーポン管理**（受付の停止/再開）
- **特集・キャンペーン管理**（トップ掲載の公開/非公開）
- **レビュー管理**（投稿レビューの非表示/再表示）
- **注文管理**（ステータス別タブ・注文番号検索・CSVエクスポート・発送状況更新）

### その他
- お気に入り（一括カート投入）
- **税込／税抜 表示切替**（全体設定）
- お問い合わせフォーム（受付番号発行）
- モバイル対応（ハンバーガードロワー）
- **SEO / PWA**：商品・パンくず JSON-LD 構造化データ・OGP・動的アイコン・`sitemap.xml`・`robots.txt`・`manifest`

## 開発

```bash
pnpm install
pnpm dev          # http://localhost:3200
pnpm build
pnpm typecheck
pnpm test
```

> npm / yarn でも可（`npm install && npm run dev` 等）。

## 主なディレクトリ

| パス | 内容 |
|------|------|
| `src/lib/catalog.ts` | カテゴリ・商品データ、サブカテゴリ分類、検索/抽出 |
| `src/lib/cart.ts` | カート状態・送料/合計計算（純関数・テスト付き） |
| `src/lib/coupons.ts` / `orders.ts` / `addresses.ts` | クーポン・注文・住所録 |
| `src/lib/features.ts` / `sort.ts` / `reviews.ts` | 特集・並び替え・レビュー生成 |
| `src/lib/overrides.ts` / `admin-stats.ts` | 管理画面の在庫/価格オーバーレイ・集計 |
| `src/components/*-provider.tsx` | カート/お気に入り/比較/ポイント/価格表示/クーポンのContext |
| `src/app/` | 各ページ（App Router）＋ `sitemap.ts` / `robots.ts` / `manifest.ts` |
| `src/app/admin/` | 管理画面 |

状態管理はすべて Context + localStorage。永続キーは `kounan_*` で名前空間化しています（内部キー名。表示ブランドとは独立）。

## テスト
